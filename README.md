# PawfectMatch - Full-Stack Pet Adoption Platform

PawfectMatch is a real-world, full-stack pet adoption portal built using the MERN stack. The platform bridges the gap between pet lovers looking to adopt and pet owners or shelters wishing to list pets for adoption. It features secure authentication, dynamic animal listing management, and a robust adoption request tracking system.

## 📌 Purpose
The core purpose of **PawfectMatch** is to simplify and digitalize the pet adoption process. It provides a centralized web application where users can responsibly explore available pets, view transparent health and vaccination statuses, and safely submit adoption requests. Simultaneously, it empowers shelters and users to efficiently manage their listed pets and monitor incoming adoption applications.

---

## ✨ Key Features
* **Secure User Authentication:** Integrated with Firebase Authentication along with secure JWT (JSON Web Token) verification on the backend to handle protected user routing and private data access.
* **Dynamic Pet Catalog & Search:** Users can explore a comprehensive list of pets across various species (Dogs, Cats, Birds, Rabbits, etc.) with detailed profiles, including age, breed, gender, adoption fees, location, and description.
* **Real-Time Request Notification & Badges:** Owners see a highlighted, blinking indicator badge on their "My Listings" dashboard immediately when a pet receives a new pending adoption request.
* **Sliding Requests Drawer Modal:** A sleek, sliding drawer interface for owners to view comprehensive applicant details (Requester Name, Email, Pickup Date, Custom Message) and directly Approve or Reject applications with immediate synchronization.
* **Personalized User Dashboards:** Separate dedicated views for users to track their submitted adoption requests ("My Requests") and for pet owners/shelters to manage their posted animals ("My Listings").
* **Responsive Dark/Light UI Elements:** Built with a clean, fully responsive layout utilizing modern animated components and adaptive styling for seamless desktop and mobile experiences.

## 📦 NPM Packages Used

### Client-Side (Frontend)
* `react` & `react-dom` - Core library for building the UI components.
* `react-router-dom` - Handles client-side routing and protected routes.
* `@tanstack/react-query` - Optimizes data fetching, caching, and state synchronization.
* `axios` - For making asynchronous HTTP API requests to the backend server.
* `firebase` - Manages user authentication and secure login states.
* `framer-motion` - Powers fluid, smooth interactive animations and sliding drawer transitions.
* `lucide-react` & `react-icons` - High-quality, modern consistent vector icons.
* `react-hot-toast` - Elegant, non-blocking floating alert notifications.
* `tailwindcss` & `daisyui` - Utility-first CSS framework and component libraries for styling.
